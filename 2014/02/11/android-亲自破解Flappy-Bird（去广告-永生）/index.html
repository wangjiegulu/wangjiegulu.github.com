[android]亲自破解Flappy Bird（去广告+永生）

Wang Jie
2014-02-11

Flappy Bird, android, 破解

听说最近Flappy Bird很火，但是难度令人发指，于是怒了，亲自破解（去广告+永生）

只要不碰到地上，永远不死，直接穿过管道- -！下载地址：http://pan.baidu.com/s/1kT7bA55

注意：只支持android版本

效果图：

 

 

 

反编译后修改smali：

去广告：

修改Flappy_Bird\smali\com\dotgears\a.smali和b.smali中run()方法中对应修改为：


const/4 v1, 0x4

    invoke-virtual {v0, v1}, Lcom/google/ads/AdView;->setVisibility(I)V

 

永生（取消碰撞检测即可）：

修改Flappy_Bird\smali\com\dotgears\flappy\c.smali


.method public static a(IIIIIIII)Z
    .locals 1

    add-int v0, p0, p2

    if-lt v0, p4, :cond_0

    add-int v0, p4, p6

    if-gt p0, v0, :cond_0

    add-int v0, p1, p3

    if-lt v0, p5, :cond_0

    add-int v0, p5, p7

    if-le p1, v0, :cond_1

    :cond_0
    const/4 v0, 0x0

    :goto_0
    return v0

    :cond_1
    const/4 v0, 0x1

    goto :goto_0
.end method

改为：


.method public static a(IIIIIIII)Z
    .locals 1

    :cond_0
    const/4 v0, 0x0

    :goto_0
    return v0

    :cond_1
    const/4 v0, 0x1

    goto :goto_0
.end method
